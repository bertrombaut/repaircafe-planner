# RepairCafe Planner

WordPress plugin voor Repair Café Renkum.

## Functies

* Evenementen beheren
* Vrijwilligers aanmelden / afmelden
* 24-uursregel voor afmelden
* Expertises koppelen aan vrijwilligers
* Max vrijwilligers per expertise
* Overzicht van aanmeldingen
* Frontend weergave

## Structuur

repaircafe-planner/

* repaircafe-planner.php
* includes/

  * admin.php
  * database.php
  * repairs.php

## Installatie

1. Upload naar: wp-content/plugins/repaircafe-planner
2. Activeer plugin in WordPress
3. Maak evenementen aan
4. Stel expertises in

## Gebruik

### Beheerder

* Evenementen maken
* Expertises beheren
* Aanmeldingen bekijken

### Vrijwilliger

* Inloggen
* Aanmelden / afmelden
* Eigen aanmeldingen bekijken

## Regel

Afmelden binnen 24 uur is niet toegestaan.

## Doel

Planning van vrijwilligers eenvoudig maken voor Repair Café.

## Versiebeheer

Plugin wordt beheerd via GitHub en gedeployed naar de live website.
